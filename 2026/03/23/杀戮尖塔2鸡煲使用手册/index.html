《杀戮尖塔2》鸡煲使用手册

故障机器人（The Defect）是《杀戮尖塔2》中机制独特、上限极高的角色，核心围绕「充能球」与「集中」展开，流派丰富且玩法灵活——既可以走长线运营的球流体系，也能玩零费暴力的物理流，新手易上手、老手能秀操作。本使用手册涵盖核心机制、五大主流流派、卡组构筑、遗物选择及实战细节，帮你快速精通这个”机械怪物”，轻松通关高进阶。

一、核心机制解析（必懂！奠定所有流派基础）

故障机器人的所有玩法都围绕「充能球」和「集中」两大核心，理解这两个机制，就能吃透80%的操作逻辑，避免走弯路。

1. 充能球（Orb）—— 核心战斗引擎

故障机器人初始拥有3个球栏，每场战斗会通过初始遗物「破损核心」自动生成1个闪电球，后续可通过卡牌生成更多球种。球的核心规则的是：球栏满后再产球，最右侧的球会被触发（消耗），触发时会释放强力主动效果，未触发时则提供持续被动加成。

四大核心球种（必记特性，流派关键）：

  • 闪电球：被动（回合结束）→ 随机攻击1名敌人；触发 → 单体高额电击伤害（电球流核心）。
  • 冰霜球：被动（回合结束）→ 获得格挡；触发 → 瞬间获得大额格挡（防御流核心）。
  • 黑暗球：被动（回合结束）→ 自身永久增加伤害；触发 → 攻击血量最低的敌人（黑球核弹流核心，叠伤越久伤害越高）。
  • 等离子球：被动（回合结束）→ 获得1点能量；触发 → 瞬间获得2点能量（能量流核心，无限能量的关键）。

2. 集中（Focus）—— 球效果放大器

集中是故障机器人的专属属性，每1点集中可使所有充能球的「被动效果」和「触发效果」提升1倍（比如1点集中让闪电球被动伤害+1、触发伤害+1，冰霜球格挡同理）。

核心增集中卡牌（所有流派通用）：碎片整理、偏差认知、耗尽、重编程——前三者直接增加集中，重编程可灵活转化属性（优先转集中/力量），拿到必留、优先升级。

3. 基础属性与初始配置

  • 初始生命值：75点（生存能力中等，需靠卡组/球效弥补）。
  • 初始遗物：破损核心（每场战斗开局生成1个闪电球，保底输出保障）。
  • 初始卡牌：打击×4、防御×4、电击、双重施放（前期优先删除打击/防御，避免稀释卡组）。

二、五大主流流派（新手→进阶，按需选择）

故障机器人流派分为「球流」和「物理流」两大类，其中球流占4种，物理流1种。新手优先从电球流、冰球流入门，熟练后再挑战黑球流、等离子流，爪击流适合喜欢暴力爽感的玩家。

流派1：电球流（最稳·新手首选）

核心思路：堆集中+量产闪电球，靠电动力学实现AOE清场，配合偏差认知、雷霆打击收尾，兼顾输出与生存，容错率极高，适合新手开荒高进阶。

▶ 核心卡牌（必拿/优先升级）

  • 产球类：雷暴、暴风雨、静电释放、电动力学（AOE产球+伤害，核心清场卡）。
  • 集中类：碎片整理、偏差认知、耗尽（优先升级，集中拉满才能最大化电球伤害）。
  • 触发/爆发类：双重释放、多重释放（触发多个电球，瞬间爆发）、雷霆打击（单体秒杀补刀）。
  • 防御类：冰川、冷静头脑、硬化机体（前期苟活，避免被秒）。

▶ 构筑要点

  1. 卡组厚度控制在20-25张，优先删除初始打击、防御，后期可删低费废牌，保证核心牌上手率。
  2. 前期（1-2层）：优先拿冰球/格挡卡苟活，遇到雷暴、碎片整理直接拿，不贪多牌。
  3. 中期（3-4层）：重点堆集中，补电球产牌，电动力学优先升级（AOE伤害翻倍）。
  4. 后期（5层+）：电动力学+多重释放清屏，偏差认知秒BOSS，冰霜球补格挡防猝死。

▶ 实战技巧

遇到多怪场景，先打电动力学产球，再用双重释放触发，瞬间清掉小怪；遇到精英/BOSS，先堆集中，再用雷霆打击+偏差认知收尾，避免恋战。

流派2：冰球防御流（苟活之王·稳过高层）

核心思路：堆集中+无限生成冰霜球，靠卡尺保留格挡，全程维持高防御，用黑球/电球补充输出，几乎不掉血，适合应对高攻高压场景（如进阶10+）。

▶ 核心卡牌（必拿/优先升级）

  • 产球类：冰川、冰寒、冷静头脑、寒流（量产冰霜球，核心防御来源）。
  • 集中类：碎片整理、偏差认知、耗尽（集中越高，冰霜球格挡越多）。
  • 防御类：硬化机体、堆栈、飞跃、充电（强化格挡，叠加防御层数）。
  • 输出类：漆黑、递归、循环（黑球叠伤，用于秒杀精英/BOSS，补输出短板）。

▶ 构筑要点

  1. 卡尺是核心遗物（必拿），可保留回合结束后的格挡，实现”无限格挡”闭环。
  2. 卡组以低费产球、格挡卡为主，少拿高费牌，避免卡手。
  3. 后期可带1-2张黑球卡，递归保黑球、循环叠伤，触发时秒杀关键目标，不用依赖电球输出。

流派3：黑球核弹流（上限最高·一击秒杀）

核心思路：堆集中+少量黑球，靠递归/循环保留黑球、无限叠伤，配合裂变+、多重释放触发，一击清场，爽感拉满，但操作难度较高，适合进阶玩家。

▶ 核心卡牌（必拿/优先升级）

  • 产球类：漆黑、彩虹、愁云惨淡、混沌（生成黑球，优先漆黑+）。
  • 叠伤类：递归、循环、漆黑+（递归保黑球不被消耗，循环叠加黑球被动伤害）。
  • 触发类：裂变+、双重释放、多重释放（瞬间触发黑球，爆发巨额伤害）。
  • 集中类：碎片整理、偏差认知、耗尽（集中拉满，黑球伤害翻倍）。

▶ 构筑要点

  1. 球栏控制在3-4个（越少越好），黑球放在最右侧，避免被其他球顶掉。
  2. 卡组要薄（18-22张），优先删废牌，保证递归、循环、漆黑上手率。
  3. 前期苟活，中期堆集中+保黑球，后期集中拉满后，用裂变+触发黑球，秒杀BOSS（单发黑球伤害可达100+）。

流派4：等离子能量流（无限能量·爽感拉满）

核心思路：量产等离子球+回能牌，实现无限能量，配合克隆子程序+、陀螺仪无限刷牌/能力，能量溢出后全技能拉满，清场效率极高。

▶ 核心卡牌（必拿/优先升级）

  • 产球类：等离子、能量注入、充电（量产等离子球，核心回能来源）。
  • 回能类：克隆子程序+、陀螺仪、能量电池（克隆子程序+是核心，可无限刷回能）。
  • 能力类：重编程、硬化机体、静电释放（强化生存与输出，利用无限能量无限释放）。
  • 爆发类：雷暴、偏差认知、超能光束（能量溢出后，瞬间清场）。

▶ 构筑要点

  1. 前期优先拿等离子球+回能卡，克隆子程序+遇到必拿（升级后效果翻倍）。
  2. 中期用陀螺仪、能量电池补充回能，确保能量循环为正（打出的牌消耗≤回能）。
  3. 后期能量溢出，可无限释放雷暴、偏差认知，清场速度碾压其他流派，注意补少量防御卡防猝死。

流派5：爪击物理流（零费暴力·不玩球）

核心思路：放弃充能球机制，堆爪击（0费，每用一次所有爪击+2伤）+过牌，实现无限零费爆发，卡组越薄越强力，适合讨厌玩球、喜欢无脑输出的玩家。

▶ 核心卡牌（必拿/优先升级）

  • 核心输出：爪击（3-5张，必拿，升级后每用一次+3伤，后期单张可达30+伤）。
  • 循环类：万物一心、刮削、精简改良（万物一心回手零费牌，刮削过牌+检索零费牌）。
  • 过牌类：FTL、全息影像、弹回、寻求（保证一回合抽完卡组，实现无限循环）。
  • 防御类：硬化机体、堆栈、飞跃（补生存，避免被高攻怪秒）。
  • 爆发类：超能光束、暴涨（叠加爪击伤害，瞬间清场）。

▶ 构筑要点

  1. 卡组越薄越好（15-20张），优先删除所有非零费废牌、诅咒牌，保证爪击、万物一心上手率。
  2. 前期拿到2张以上爪击即可走此流派，刮削、万物一心遇到必拿（刮削优先升级）。
  3. 后期靠刮削+万物一心无限刷爪击，爪击伤害叠到30+后，0费无限甩，BOSS血条直接蒸发。

三、卡组构筑通用原则（所有流派适用）

  1. 卡组厚度：控制在18-25张，流派不同略有差异（爪击流、黑球流更薄，冰球流可稍厚），优先删打击、防御、诅咒，避免稀释核心牌。
  2. 费用曲线：以0费/1费为主（产球、过牌、爪击），2-3费为辅（集中、爆发、防御），尽量不拿4费+卡牌，避免卡手。
  3. 三要素必带：① 输出/产球（对应流派核心牌）；② 集中/增伤（碎片整理、偏差认知等通用卡）；③ 防御/过牌（保证生存与循环）。
  4. 升级优先级：核心产球/输出卡（雷暴、冰川、爪击）＞集中卡（碎片整理、偏差认知）＞循环/过牌卡（刮削、万物一心）＞防御卡。

四、核心遗物推荐（按优先级排序）

遗物是故障机器人的”质变关键”，不同流派对应不同最优遗物，以下是通用+流派专属遗物，遇到必拿。

1. 通用必拿遗物（所有流派适用）

  • 奇点：每回合开始时，若手牌为0，抽2张牌，大幅提升过牌效率，适配所有流派。
  • 咖啡滤杯：每回合多1点能量，无法睡觉回血（无限流/暴力流可无视回血），提升启动速度。
  • 能量电池：开局稳定上手2张零费卡，避免核心卡沉底，适合所有流派前期过渡。
  • 蓝蜡烛：可打出费用高于当前能量的牌（扣血），启动阶段救急，适合所有流派。

2. 流派专属遗物

  • 电球流：加特林（增加攻击频率）、避雷针（提升电球伤害）。
  • 冰球流：卡尺（保留格挡，核心质变）、自形黏土（掉血加格挡，提升生存）。
  • 黑球流：转轮（每回合随机获得技能牌）、黑暗石（提升黑球叠伤速度）。
  • 等离子流：陀螺仪（无限回能）、能量核心（提升等离子球效果）。
  • 爪击流：拳刃（给爪击附魔动量，叠伤更快）、音乐盒（复制爪击/万物一心）。

五、实战技巧与避坑指南（大神私藏）

1. 通用实战技巧

  • 启动顺序：先打产球/过牌卡，再打集中/回能卡，最后打触发/爆发卡，避免卡手（比如先打编译错误，再打产球卡，会浪费回能效果）。
  • 球栏管理：球流尽量保持球栏有1个空位，避免新球顶掉核心球（黑球流除外，需控制球栏数量）。
  • 删牌优先级：诅咒＞初始打击/防御＞低费废牌＞非核心高费牌，后期可删重复的辅助卡。

2. 各流派避坑点

  • 电球流：别贪多球，集中不够时，多球反而稀释伤害，优先堆集中再补球。
  • 冰球流：别只堆防御不补输出，后期需带1-2张黑球/电球，避免打BOSS太慢。
  • 黑球流：别堆太多黑球，3-4个球栏+1张黑球即可，多了反而容易被顶掉，无法叠伤。
  • 等离子流：别拿太多高费牌，确保能量循环为正，否则会出现”有能量但没牌打”的情况。
  • 爪击流：别贪多爪击，3-5张足够，多了会稀释过牌，反而影响循环。

3. 高进阶应对技巧

进阶10+后，敌人伤害高、机制复杂，重点注意：① 优先拿防御卡/遗物，避免猝死；② 针对高攻BOSS（如 Awakened One），优先堆冰霜球/格挡；③ 针对高血BOSS（如 Time Eater），优先堆黑球/电球爆发；④ 遇到禁锢怪（限制出牌），带1张解控卡，避免循环断裂。

六、总结（快速上手指南）

故障机器人的核心是「灵活适配」——开局看核心卡/遗物，哪套来得多就走哪条路，不用硬凑流派。新手从电球流、冰球流入门，熟悉充能球与集中机制后，再挑战黑球流、等离子流，爪击流可作为”备用流派”（开局爪击多可走）。

记住：卡组不在于多，而在于精；集中是球流的灵魂，循环是暴力流的核心，只要抓好这两点，你就能轻松用故障机器人通关所有进阶，成为尖塔大神！
